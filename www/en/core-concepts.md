# Core Concepts

## Modes

- `Build`: default mode for implementing changes.
- `Plan`: structured planning for larger tasks.

## Session

A session stores your full conversation and can be resumed later.

## Approval Policy

Controls when file edits and shell commands require your approval.

## Iteration Budget

`max_iterations` limits how many tool rounds the agent can execute per task.

## Skills

Skills add domain-specific instructions for focused workflows.
