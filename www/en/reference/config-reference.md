# Config Reference

Common fields in `config.json`:

- `provider`: model provider settings
- `approval_policy`: approval behavior for risky actions
- `max_iterations`: max tool rounds per task
- `stream`: streaming response control
- `update_check.enabled`: update check toggle

See [`config.example.json`](https://github.com/1024XEngineer/bytemind/blob/main/config.example.json) for a complete sample.
